# 2048 Game

A React implementation of the classic 2048 puzzle game.
## Installation
1. Clone the repository
2. Install dependencies:
```bash
    npm install
```
#. Start the development server:
```bash
    npm start
```
4. Open the link to play the game

## How to Play

Use your arrow keys to move the tiles. When two tiles with same number touch, they merge into one. Try to reach 2048!
## Project Structure 
- `/src/components` - React components
- `/src/utils` - Game logic utilities
- `/src/constants` - Configuration and constants
- `/src/styles` -CSS animations

## Build
To create a production build:
```Bash
nmp run build
```
