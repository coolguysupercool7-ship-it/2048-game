## Installation

2. Install dependencies:

## How to Play

Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one. Try to reach 2048!

## Project Structure

- `/src/components` - React components
- `/src/helpers` - Game logic utilities
- `/src/config` - Configuration and constants
- `/src/styles` - CSS animations
